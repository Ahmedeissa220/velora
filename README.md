## 🧠 About The Project

Velora is a **modern luxury e-commerce platform** designed to simulate real-world shopping experiences with a premium UI and smooth UX.

The project focuses on:
- Clean architecture
- Reusable components
- State management using Context API
- High-performance UI with Vite

---

## ✨ Features

- 🛍️ Product Catalog System
- ❤️ Wishlist (Add / Remove Favorites)
- 🛒 Shopping Cart System
- ⚡ Lightning-fast performance
- 🎨 Luxury Glassmorphism UI
- 📱 Fully Responsive Design
- 🔄 Smooth navigation with React Router

---

## 🧱 Tech Stack

- React.js (Frontend)
- Vite (Build Tool)
- Tailwind CSS (Styling)
- React Router DOM (Routing)
- Context API (State Management)
- Lucide Icons
